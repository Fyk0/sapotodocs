Ticket Commands

 Command  Usage  Aliases  Example
 add  Gives the specified user permissions to view the ticket. This can only be ran in a ticket channel.  a  add
 close  Closes the current ticket. This can only be ran in a ticket channel.  c  close
 new  Creates a new ticket channel and gives permissions to the configured support role and the ticket creator to read and write.  n  new [subject]
 ticket  This pulls information on the ticket such as creator, when it was opened, how many messages were sent, etc.  t  ticket [--log]


Configuration Commands

 Command  Usage  Aliases  Example
 blacklist  Adds a user to the servers ticket blacklist.  block  blacklist
 config  Lists the servers configuration  config-list  config
 force-captcha  PREMIUM Force users to solve a captcha on their next ticket even if they've never made one before.  fc  force-captcha
 set-category  Changes the server's ticket category.  category  set-category
 set-log  Changes this guild's ticket log.  log  set-log
 set-max  Changes the maximum amount of tickets a user can have open at once.  max  max
 set-welcome  Changes this guild's welcome channel.  welcome  welcome
 set-prefix  Changes the servers prefix. The prefix cannot have spaces and more than 8 characters.  prefix  prefix
 set-msg  PREMIUM Changes this guild's new ticket message.  msg  set-msg
 set-support  Changes this guild's Support Team role.  support  set-support
 set-welcome-msg  Changes this guild's welcome message.  welcome-msg  set-welcome-msg
  • %user% - user mention
  • %username% - user's username
  • %tag% - user's tag
  • %server - server name
  • %server.size% - server member count
  • %ticket_channel% - all whitelisted channels joined on or PREMIUM
 toggle-captcha  PREMIUM Toggle is the user creating a ticket must solve a captcha to create a ticket.  N/A  toggle-captcha
 toggle-delete  PREMIUM Toggle if the new-ticket execution message is deleted or not. This can prevent other users from copying confidential data.  N/A  toggle-delete
 toggle-welcome  Toggle the new-member welcome message.  N/A  toggle-welcome
 whitelist  Restricts users to making tickets in the whitelisted channels.  wl  whitelist
